## 🎮 Keyboard Shortcuts Reference

### Accessibility Control
- **Alt+A**: Open/close accessibility panel
- **Alt+Plus/Minus**: Increase/decrease text size
- **Alt+C**: Cycle contrast themes
- **Alt+D**: Toggle dark mode
- **Alt+I**: Invert colors
- **Alt+G**: Toggle grayscale

### Text-to-Speech
- **Alt+T**: Toggle text-to-speech
- **Alt+R**: Start/stop reading page
- **Alt+[**: Decrease speech rate
- **Alt+]**: Increase speech rate

### Navigation
- **H / Shift+H**: Next/previous heading
- **1-6**: Next heading of that level
- **L / Shift+L**: Next/previous landmark
- **T / Shift+T**: Next/previous table
- **F / Shift+F**: Next/previous form field
- **M / Shift+M**: Next/previous list

### Reading Aids
- **Alt+U**: Toggle reading ruler
- **Alt+K**: Toggle reading mask
- **Alt+S**: Toggle reading spotlight

### Other
- **Escape**: Exit virtual mouse / close panels
- **Ctrl+P** (in PDF mode): Export to PDF

## 🚀 Technical Implementation

### Technologies Used
- **React Context API**: Global state management
- **Web Speech API**: Text-to-speech and voice commands
- **Web Speech Recognition API**: Voice command processing
- **CSS Filters**: Visual effects (grayscale, sepia, etc.)
- **Request Animation Frame**: Smooth 60fps animations
- **LocalStorage**: Persistent settings across sessions
- **MutationObserver**: Dynamic content monitoring
- **Intersection Observer**: Performance optimizations

### Performance
- **Lazy loading**: Managers only active when features enabled
- **Debounced updates**: Prevent excessive re-renders
- **Memoized callbacks**: Optimized event handlers
- **CSS-based effects**: Hardware-accelerated transforms

### Browser Support
- **Chrome/Edge**: Full support (all 101 features)
- **Firefox**: Full support (all 101 features)
- **Safari**: Partial voice command support
- **Mobile browsers**: Touch-optimized interactions

## 📝 Usage Examples

### Enable High Contrast for Low Vision
1. Click accessibility widget (floating button)
2. Go to "Vision & Display" tab
3. Toggle "High Contrast"
4. Select desired contrast theme

### Navigate Page with Keyboard Only
1. Press `Alt+A` to open accessibility panel
2. Enable "Keyboard Navigation"
3. Use `H` to jump between headings
4. Use `Tab` to focus interactive elements
5. Press `Enter` to activate

### Use Voice Commands
1. Enable "Voice Commands" in Motor & Dexterity tab
2. Say "increase text size" or "enable dark mode"
3. See confirmation in bottom-right indicator
4. Say "open accessibility menu" to adjust settings

### Export Accessible PDF
1. Enable "PDF Export Mode" in Advanced Tools
2. Review optimized layout
3. Click "Print/Export" or press `Ctrl+P`
4. Select "Save as PDF" in print dialog
5. PDF includes accessibility metadata
